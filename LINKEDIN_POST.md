🛠️ **Deep Dive: Building a Real-time System Resource Monitor**

I recently completed **SysMon**, a full-stack application designed to provide real-time telemetry for system resources. This project allowed me to explore several complex technical challenges:

✅ **Live Data Streaming:** Implemented full-duplex communication using Django Channels, allowing system metrics (collected via `psutil`) to be pushed to the browser every second.
✅ **Scalable Infrastructure:** Dockerized the entire stack and moved from manual deployment to a robust Kubernetes setup using Kustomize and KinD.
✅ **Data Visualization:** Built dynamic, responsive gauges and history charts using Recharts and React.
✅ **Resilient Proxying:** Configured Nginx to handle both static content and WebSocket handshakes efficiently within a containerized environment.
